# nullsteuer.llc Approved Americana Assets

This public repository contains the **15 approved Kodachrome Americana source assets** for nullsteuer.llc plus deterministic 4:5 card crops derived from those files.

The original compositions were not regenerated. Landscape files are optimized WebP images, the badge is a transparent PNG, and card crops use the `-card.webp` suffix at a 4:5 aspect ratio.

Website code should reference immutable raw GitHub URLs pinned to a commit SHA rather than branch-relative URLs.
